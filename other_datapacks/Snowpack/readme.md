# 积雪
原版下雪天的雪片只有一层，本模组去掉了这种限制，雪片可以无限堆，堆到8层会得到雪块并可以继续往上堆。(实际上速度很慢，不用担心被雪淹没不知所措)
使用命令
```scoreboard players set #period snowpackvalue [数值]```
来修改可堆叠的雪层生成速度，数值越高速度越慢，默认为4。

将snowpack.zip放入存档datapacks文件夹即可。
